CSS কি?

CSS (Cascading Style Sheets) হলো একটি স্টাইলিং ভাষা, যা HTML উপাদানগুলির সৌন্দর্য ও বিন্যাস নিয়ন্ত্রণ করে।

p { color: blue; }

এই টেক্সট নীল রঙের।

কেন CSS ব্যবহার করা হয়?

HTML কনটেন্টকে সুন্দর, আকর্ষণীয় ও ব্যবহারকারী-বান্ধব করতে CSS ব্যবহৃত হয়।

h1 { text-align: center; color: green; }

CSS শেখা মজা!

CSS এর ধরন

CSS তিনভাবে যুক্ত করা যায়:

  • Inline CSS
  • Internal CSS
  • External CSS
<p style="color:red;">Inline CSS</p>

Inline CSS

CSS এর গঠন

CSS কোড মূলত দুটি অংশে বিভক্ত: সিলেক্টর এবং ডিক্লারেশন ব্লক।

selector { property: value; }

উদাহরণ: h1 { color: blue; }

CSS সিলেক্টর সমূহ

CSS সিলেক্টর দ্বারা HTML এলিমেন্ট নির্বাচন করা হয়। যেমন:

  • Element Selector
  • Class Selector
  • ID Selector
  • Universal Selector
.className { color: pink; }

Class Selector Example

CSS এর গুরুত্বপূর্ণ প্রপার্টি

যেমন: color, background-color, font-size, margin, padding ইত্যাদি।

div { background-color: lightblue; padding: 20px; }
Background সহ ডিভ।

Box Model

HTML প্রতিটি এলিমেন্ট একটি বক্স হিসেবে বিবেচিত হয়: Margin, Border, Padding এবং Content।

div { padding: 20px; border: 5px solid black; margin: 10px; }
Box Model Example

Display প্রপার্টি

HTML এলিমেন্ট কিভাবে প্রদর্শিত হবে তা নির্ধারণ করে (inline, block, flex, grid ইত্যাদি)।

span { display: block; } Block Span

Position প্রপার্টি

এলিমেন্টের পজিশন নির্ধারণ করতে ব্যবহৃত হয়: static, relative, absolute, fixed, sticky।

div { position: relative; top: 10px; }
Relative Position

Flexbox ও Grid Layout

CSS Flexbox এবং Grid Layout এলিমেন্টের লেআউট ডিজাইন করতে ব্যবহৃত হয়।

display: flex;
Flex 1
Flex 2

Pseudo-class ও Pseudo-element

বিশেষ অবস্থায় এলিমেন্ট স্টাইল করার জন্য ব্যবহৃত হয়, যেমন :hover, :first-child, ::before, ::after ইত্যাদি।

a:hover { color: red; } মাউস নিয়ে আসুন

Transition ও Animation

এলিমেন্টে ধীরে ধীরে পরিবর্তন বা মুভমেন্ট যোগ করা হয়।

div { transition: all 0.5s; }
Hover me!

Media Queries

ভিন্ন ভিন্ন ডিভাইস সাইজ অনুযায়ী CSS স্টাইল পরিবর্তন করতে ব্যবহৃত হয়।

@media (max-width: 600px) { body { background: yellow; } }

স্ক্রীন ছোট করলে ব্যাকগ্রাউন্ড পরিবর্তন হবে।

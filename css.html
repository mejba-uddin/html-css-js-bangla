CSS সম্পূর্ণ গাইড

CSS এর পুরো নাম হলো ক্যাসকেডিং স্টাইল শীটস (Cascading Style Sheets)। এটি HTML উপাদানগুলোর স্টাইল বা ডিজাইন ঠিক করার জন্য ব্যবহৃত হয়।

CSS কি কি নিয়ন্ত্রণ করে?

  • টেক্সটের রং ও স্টাইল (color, font, text-align)
  • ব্যাকগ্রাউন্ড (background-color, background-image)
  • বর্ডার (border, border-radius)
  • স্পেসিং (margin, padding)
  • আকার ও অবস্থান (width, height, position)
  • ডিসপ্লে ও লেআউট (display, flexbox, grid)
  • এনিমেশন ও ট্রানজিশন (animation, transition)
  • রেসপনসিভ ডিজাইন (media queries)

CSS লেখার নিয়ম

CSS লেখার সাধারণ গঠন:

নির্বাচক {
প্রপার্টি: মান;
}

CSS এর কিছু গুরুত্বপূর্ণ প্রপার্টি

  • color - লেখা বা টেক্সটের রং নির্ধারণ করে।
  • background-color - ব্যাকগ্রাউন্ডের রং নির্ধারণ করে।
  • font-size - লেখার আকার নির্ধারণ করে।
  • margin - উপাদানের বাইরের ফাঁকা জায়গা নির্ধারণ করে।
  • padding - উপাদানের ভেতরের ফাঁকা জায়গা নির্ধারণ করে।
  • border - উপাদানের চারপাশে সীমারেখা তৈরি করে।
  • width এবং height - উপাদানের প্রস্থ ও উচ্চতা নির্ধারণ করে।
  • text-align - লেখা কিভাবে সজ্জিত হবে (বামে, ডানে, মাঝে) তা নির্ধারণ করে।
  • display - উপাদানের প্রদর্শন নিয়ন্ত্রণ করে (যেমন block, inline, flex)।
  • position - উপাদান কোথায় অবস্থিত হবে তা নির্ধারণ করে (static, relative, absolute, fixed)।
  • flex এবং grid - লেআউট ডিজাইন করার জন্য ব্যবহৃত হয়।
  • animation ও transition - মুভমেন্ট ও পরিবর্তন যোগ করতে ব্যবহৃত হয়।
  • media queries - ভিন্ন ভিন্ন ডিভাইসে কিভাবে স্টাইল হবে তা নির্ধারণ করে।

CSS যুক্ত করার ৩টি উপায়

  • Inline CSS - সরাসরি HTML ট্যাগের মধ্যে লিখা।
  • Internal CSS - <style> ট্যাগের মধ্যে লিখা।
  • External CSS - আলাদা .css ফাইলে লিখে HTML ফাইলে লিংক করা।

উদাহরণসমূহ

Inline CSS

<h1 style="color: red;">হ্যালো বিশ্ব</h1>

Internal CSS

<style>
p { color: green; }
</style>

External CSS

<link rel="stylesheet" href="style.css">

উন্নত CSS বিষয়বস্তু

  • Flexbox: উপাদানগুলোকে এক লাইনে সাজানোর সহজ উপায়।
  • Grid: জটিল লেআউট তৈরির জন্য ব্যবহৃত হয়।
  • Responsive Design: বিভিন্ন স্ক্রিন সাইজের জন্য ডিজাইন তৈরি।
  • Variables: CSS এ ভেরিয়েবল ব্যবহার করে কোড সংক্ষিপ্ত ও উন্নত করা।
  • Custom Fonts: নিজের পছন্দের ফন্ট যোগ করা।

উপসংহার

CSS শেখা যতই গভীরে যাবেন, ততই মজার হয়ে উঠবে। নিয়মিত প্র্যাকটিস করুন এবং নিজের প্রোজেক্ট তৈরি করে শিখুন।

CSS সম্পূর্ণ গাইড

১. CSS কি?

CSS এর পুরো নাম হলো ক্যাসকেডিং স্টাইল শীটস (Cascading Style Sheets)। এটি HTML উপাদানগুলোর স্টাইল বা ডিজাইন ঠিক করার জন্য ব্যবহৃত হয়।

২. কেন CSS ব্যবহার করা হয়?

  • ওয়েবপেজকে সুন্দর এবং আকর্ষণীয় দেখানোর জন্য।
  • HTML থেকে আলাদা করে স্টাইল মেইনটেইন করার জন্য।
  • একাধিক পেজে একই স্টাইল প্রয়োগ করার জন্য।

১০. Flexbox ও Grid Layout

Flexbox উদাহরণ:

.container { display: flex; justify-content: center; align-items: center; }

Grid উদাহরণ:

.container { display: grid; grid-template-columns: 1fr 1fr; gap: 10px; }

১১. Pseudo-class ও Pseudo-element

  • Pseudo-class: যেমন :hover, :focus, :nth-child()
  • Pseudo-element: যেমন ::before, ::after

উদাহরণ:

button:hover { background-color: blue; color: white; }

১২. Transition এবং Animation

Transition উদাহরণ:

div { transition: all 0.3s ease; }

Animation উদাহরণ:

@keyframes example { from {background-color: red;} to {background-color: yellow;} } div { animation: example 5s infinite; }

১৩. Media Queries

@media (max-width: 600px) { body { background-color: lightblue; } }

উপসংহার

CSS শেখা যতই গভীরে যাবেন, ততই মজার হয়ে উঠবে। নিয়মিত প্র্যাকটিস করুন এবং নিজের প্রোজেক্ট তৈরি করে শিখুন। নিজে নিজে বিভিন্ন ডিজাইন বানিয়ে প্র্যাকটিস করুন!

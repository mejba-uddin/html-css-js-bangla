CSS কি?

CSS (Cascading Style Sheets) হলো একটি স্টাইলিং ভাষা, যা HTML উপাদানগুলির সৌন্দর্য ও বিন্যাস নিয়ন্ত্রণ করে। CSS ব্যবহার করে আমরা ওয়েব পেজের ডিজাইন এবং লেআউটকে আরও আকর্ষণীয় করে তুলতে পারি।

p { color: blue; }

এই টেক্সট নীল রঙের।

কেন CSS ব্যবহার করা হয়?

HTML কনটেন্টকে সুন্দর, আকর্ষণীয় ও ব্যবহারকারী-বান্ধব করতে CSS ব্যবহৃত হয়। CSS এর মাধ্যমে আমরা বিভিন্ন ডিজাইন উপাদান যেমন রঙ, ফন্ট, মার্জিন, প্যাডিং ইত্যাদি নিয়ন্ত্রণ করতে পারি।

h1 { text-align: center; color: green; }

CSS শেখা মজা!

CSS এর ধরন

CSS তিনভাবে যুক্ত করা যায়:

  • Inline CSS: HTML ট্যাগের মধ্যে সরাসরি স্টাইল যুক্ত করা হয়।
  • Internal CSS: HTML ডকুমেন্টের <head> সেকশনে <style> ট্যাগের মধ্যে CSS কোড লেখা হয়।
  • External CSS: একটি আলাদা CSS ফাইল তৈরি করে সেটিকে HTML ডকুমেন্টে <link> ট ্যাগের মাধ্যমে যুক্ত করা হয়।
<p style="color:red;">Inline CSS</p>

Inline CSS

CSS এর গঠন

CSS কোড মূলত দুটি অংশে বিভক্ত: সিলেক্টর এবং ডিক্লারেশন ব্লক। সিলেক্টর দ্বারা আমরা কোন HTML এলিমেন্টকে টার্গেট করছি তা নির্ধারণ করি এবং ডিক্লারেশন ব্লকে প্রপার্টি ও তাদের মান উল্লেখ করা হয়।

selector { property: value; }

উদাহরণ: h1 { color: blue; }

CSS সিলেক্টর সমূহ

CSS সিলেক্টর দ্বারা HTML এলিমেন্ট নির্বাচন করা হয়। যেমন:

  • Element Selector: নির্দিষ্ট HTML এলিমেন্ট নির্বাচন করে।
  • Class Selector: একটি বা একাধিক এলিমেন্টের জন্য একই স্টাইল প্রয়োগ করতে ব্যবহৃত হয়।
  • ID Selector: একটি নির্দিষ্ট এলিমেন্টের জন্য স্টাইল প্রয়োগ করতে ব্যবহৃত হয়।
  • Universal Selector: সব এলিমেন্টকে টার্গেট করে।
.className { color: pink; }

Class Selector Example

CSS এর গুরুত্বপূর্ণ প্রপার্টি

CSS এর কিছু গুরুত্বপূর্ণ প্রপার্টি হলো: color, background-color, font-size, margin, padding ইত্যাদি। এই প্রপার্টিগুলি ব্যবহার করে আমরা এলিমেন্টের চেহারা পরিবর্তন করতে পারি।

div { background-color: lightblue; padding: 20px; }
Background সহ ডিভ।

Box Model

HTML প্রতিটি এলিমেন্ট একটি বক্স হিসেবে বিবেচিত হয়: Margin, Border, Padding এবং Content। Box Model বুঝতে পারলে আমরা এলিমেন্টের সঠিক বিন্যাস ও স্থান নির্ধারণ করতে পারি।

div { padding: 20px; border: 5px solid black; margin: 10px; }
Box Model Example

Display প্রপার্টি

HTML এলিমেন্ট কিভাবে প্রদর্শিত হবে তা নির্ধারণ করে (inline, block, flex, grid ইত্যাদি)। Display প্রপার্টি ব্যবহার করে আমরা এলিমেন্টের প্রদর্শন পদ্ধতি পরিবর্তন করতে পারি।

span { display: block; } Block Span

Position প্রপার্টি

এলিমেন্টের পজিশন নির্ধারণ করতে ব্যবহৃত হয়: static, relative, absolute, fixed, sticky। Position প্রপার্টি ব্যবহার করে আমরা এলিমেন্টের অবস্থান নিয়ন্ত্রণ করতে পারি।

div { position: relative; top: 10px; }
Relative Position

Flexbox ও Grid Layout

CSS Flexbox এবং Grid Layout এলিমেন্টের লেআউট ডিজাইন করতে ব্যবহৃত হয়। Flexbox একমাত্র দিকের লেআউটের জন্য এবং Grid দুইটি দিকের লেআউটের জন্য উপযুক্ত।

display: flex;

Flex 2

Pseudo-class ও Pseudo-element

বিশেষ অবস্থায় এলিমেন্ট স্টাইল করার জন্য ব্যবহৃত হয়, যেমন :hover, :first-child, ::before, ::after ইত্যাদি। Pseudo-class ব্যবহার করে আমরা এলিমেন্টের বিশেষ অবস্থার জন্য স্টাইল প্রয়োগ করতে পারি।

a:hover { color: red; } মাউস নিয়ে আসুন

Transition ও Animation

এলিমেন্টে ধীরে ধীরে পরিবর্তন বা মুভমেন্ট যোগ করা হয়। Transition ব্যবহার করে আমরা এলিমেন্টের স্টাইল পরিবর্তনের সময়কাল নির্ধারণ করতে পারি।

div { transition: all 0.5s; }
Hover me!

Media Queries

ভিন্ন ভিন্ন ডিভাইস সাইজ অনুযায়ী CSS স্টাইল পরিবর্তন করতে ব্যবহৃত হয়। Media Queries ব্যবহার করে আমরা রেসপন্সিভ ডিজাইন তৈরি করতে পারি।

@media (max-width: 600px) { body { background: yellow; } }

স্ক্রীন ছোট করলে ব্যাকগ্রাউন্ড পরিবর্তন হবে।
